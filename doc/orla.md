# Orla

* https://github.com/dorcha-inc/orla
* doc : https://orlaserver.github.io/#/


## Quickstart

install
```
./aistack orla install
```

info
```
./aistack orla info
```


launch orla standalone agent
```
./aistack orla launch -- agent
```

launch orla service
```
./aistack orla launch -- serve
```

get service health
```
curl -s http://localhost:8801/api/v1/health
```
